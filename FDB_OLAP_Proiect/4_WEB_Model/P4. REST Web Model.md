## Implementare REST API si Web UIX

Implementarea este realizata peste modelul analitic din P3.

Tehnologii folosite:
- REST: FastAPI (alternativa valida la ORDS, conform observatiei din template)
- Web UIX: Streamlit (dashboard de tip report/chart)
- Baza de date: PostgreSQL (schema integration_model)

(1) Nivel/Model Web RESTful

View_Resursa_REST_1: analytics_budget_runtime
Nume/descriere:
Expune agregari pe categorii de buget si runtime.
Tabela/view analitic/de integrare tinta:
integration_model.av_budget_runtime_rollup
Detalii de implementare:
- endpoint FastAPI GET /api/v1/analytics/budget-runtime
- executie SELECT peste view-ul OLAP
fraze SQL:
SELECT * FROM integration_model.av_budget_runtime_rollup LIMIT :limit;
URL resursa RESTful:
http://localhost:8000/api/v1/analytics/budget-runtime?limit=100
Descriere concreta (JSON):
[
  {
    "budget_bucket": "high",
    "runtime_bucket": "standard",
    "movie_count": 120,
    "avg_user_rating": 3.82,
    "avg_roi_percent": 145.3,
    "total_revenue_musd": 53120.11
  }
]

## Comenzi rapide de rulare (local)

1. Instalare dependinte
pip install -r FDB_OLAP_Proiect/4_WEB_Model/P4_requirements.txt

2. Start REST API
uvicorn P4_rest_api_fastapi:app --reload --port 8000 --app-dir FDB_OLAP_Proiect/4_WEB_Model

3. Start Web UIX
streamlit run FDB_OLAP_Proiect/4_WEB_Model/P4_web_uix_streamlit.py --server.port 8501
